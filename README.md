## Sensirion SPS30 Particulate Matter Sensor driver

* [Crate](https://crates.io/crates/sps30)
* [Documentation](https://docs.rs/sps30/)
* [Usage](#usage)
* [License](#license)

## Usage

Add `sps30` to `Cargo.toml`

```toml
[dependencies]
sps30 = "^0.1.0"
```

or

```toml
[dependencies.sensirion-hdlc]
git = "https://github.com/iohe/sps30"
```
